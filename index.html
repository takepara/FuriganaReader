How to get a free API key

This app uses Google's Gemini AI to analyze Japanese text. To use it, you need a personal API key — think of it as a password that tells Google's AI “this request is from me.”

  1. Go to Google AI Studio
    aistudio.google.com/apikey
    Sign in with your Google account (Gmail, etc.).
  2. Click “Create API key”
    Choose “Create API key in new project” — this takes just a few seconds.
  3. Copy your key
    It looks like: AIzaSy...
    Paste it into the box on this page and click Save & Continue.
Is it really free?
Yes — Google offers a generous free tier for casual use. You don’t need to enter a credit card to get started.

What if I use it a lot?
If you analyze large amounts of text regularly, you may eventually hit the free limit. In that case, Google will ask you to add a payment method to continue — you only pay for what you use beyond the free tier. Typical personal use costs just a few cents per month at most.
